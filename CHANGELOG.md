# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),

## [Unreleased]

- nothing yet

## [v0.2.0] - 2025-10-23

### Added

- Quantity field for materials & expenses line items
- Dynamic product title
- Delete confirmations on saved product setups to prevent mis-clicks
- Display for total labor hours
- Tool tips for helpful information

### Changed

- Refactored logic interactions

## [v0.1.0] - 2025-10-21

### Added

- System to save your inputs so that you can easily load them again to verify or make changes
- Included feature to append line items to what you are currently working on to quickly add reused items
- Quick calculator to see estimated returns from a given listing price
- Very basic styling to get the theme started

### Changed

- Templates are now handled differently, making addItem more robust
- HTML structure is now setup to be accessible, semantic, and SEO friendly

## [v0.0.1] - 2025-10-19

### Added

- Initial tool release to calculate suitable Etsy seller listing prices based on fees
